以及这是一篇很个人向的读后感，存在拆 CP 的情况请见谅。

先说定位吧……我把它当作一本阿不思 X 斯科普斯 + 隐藏德哈的子世代同人文来看，于是收获一本满足。

原著中主角的人设没有太毁，但是也没有太大的关系。肯定和那些血气方刚的少年们不是一拨人。大家看看就好啦。

不过，看之前我心目中的阿不思·西弗勒斯·波特和斯科普斯·马尔福的形象是这样的：

看完之后发现其实是这样的：

阿不思：你知道三强争霸赛吗？
斯科普斯（超开心地）：哦哦哦哦哦哦~那是一个测验，三个学校选出三位冠军完成三个任务去赢得一个奖杯。咋啦？
阿不思：噫你好 geek 啊，你知道吗？
斯科普斯：嗯啊。

你需要知道的背景是：

阿不思·西弗勒斯·波特被分进了斯莱特林，魔法很差，在父亲的盛名之下压力巨大，在重重隔阂之下意识到父亲的成功建立在无数人的牺牲之上。

斯科普斯·马尔福被传言说是阿斯托利亚和伏地魔的孩子，被各种歧视，（虽然我不太能理解，因为我觉得当黑魔王的孩子挺酷的），他还喜欢读书，我感觉和《权力的游戏》里 Sam 的形象差不多。

这本书里我主要关注的点有三：原著梗、感(ji)情线和神奇的脑洞。

二、感情线

以及，必须要承认年少的时候我被安利了德赫这个大坑，后来又被@秣忆雨淋的《路遇寂寞之歌》安利了斯科普斯 X 罗丝……（不过书里的设定是罗丝是德拉科的孩子，这我是不接受的嗯）

五、一些零碎的感想

这本书通过时空转换器，展开了很多神奇的脑洞。我很喜欢这种 what if 的设想，《老友记》S6E15、16，《实习医生格蕾》S8E13，《武林外传》第四十回都脑补过这种桥段，充满了 YY 的色彩，看着还是很过瘾的。像是如果罗恩和赫敏没好上会怎么样，黑魔王赢了会怎么样，这些构想在书里还挺有画面感的。

剧情上，我反正看得挺开心啦。这里还有一些小想法：

1.斯莱特林的视角

看到阿不思被分进斯莱特林我还挺期待的，不光是开脑洞一个波特被分进斯莱特林会是怎样的体验，而且我还挺想看从斯莱特林的视角讲述故事。

看 7 部《哈利·波特》，我们都习惯了从格兰芬多的视角看待事物，忠诚，勇敢，无所畏惧。斯莱特林仿佛就是种族歧视 + 阴险狡诈。但是如果四个学院之一都是纯粹的坏人，那这个学校是吃枣药丸啊……

所以，我很希望这个故事从斯莱特林的视角去讲。一个根正苗红的波特，在校园生活和重重冒险中，渐渐被斯莱特林的精神感染，最终认清真正的自己，也让所有人消除偏见，相互理解。前面提到第二世中的情节：

哈利：课间你哪儿也不许去！你必须一直待在格兰芬多公共休息室！

阿不思：去不了！我是个斯莱特林！！

在哈利的心目中，霍格沃兹里最安全的地方就是暖烘烘的格兰芬多公共休息室，所以他教训孩子，下意识地就让他一直呆在那里。

但是阿不思是个斯莱特林，他的霍格沃兹是绿莹莹的、阴暗潮湿的地下室。（不过我也不喜欢这种地方，感觉太刻意黑了……斯莱特林的公共休息室应该是那种陈旧的、很古典、有品位、在细节处富丽堂皇的地方。）

按照我的理解，斯莱特林的阴险狡诈是一种表象，他们真正的精神是目标明确，且胜利至高无上。所以，他们才会为了达成目的不择手段。跟格兰芬多的勇敢无畏相比，他们遇到危险会审时度势，明哲保身；跟赫奇帕奇的勤恳踏实相比，他们又显得诡计多端。其实都是为了赢。

原著中每一个有名的斯莱特林，感觉都背负着某些负担而来。汤姆·里德尔的仇恨，斯内普的孤苦，马尔福的家族重担，以及阿不思的盛名之下其实难副，所以他们的校园生活不会像其他人那样且行且歌，他们需要动用一切力量去实现野心，达成抱负。

这些都不同于《哈利·波特》原著中所弘扬的价值。如果说格兰芬多的勇敢来源于他们的爱与忠诚。那么我觉得对斯莱特林来说，对胜利的渴望才是他们的不竭力量之源。在斯莱特林心目中，友谊是一种志同道合、共谋大业，或者说是得知我幸，没有就算了我跟自己玩。在斯莱特林的视角下，宣扬友谊的不可或缺就有一点违和……斯莱特林应该只忠于自己，无惧孤独。

（每个学院都有庸庸碌碌之辈嘛，不要跟我提克拉布和高尔啦。）

也许我对斯莱特林的理解不是完全符合罗琳的意图，但是《哈利·波特与倒霉孩子》这个故事没有足够的篇幅去阐释斯莱特林的特质，我是觉得有一点遗憾的。不过也还好，作为一个系列，它的精神内核应该是一以贯之的，如果转换视角，推翻之前建构的价值观重头再来，万一崩坏了怎么办。

另外，我一直很想看的是阿不思·波特和小天狼星·布莱克的互动。他们都是家族的“叛逆者”。对于小天狼星来说，和詹姆他们的友情是他逃离家族的宣泄口。（我非常喜欢第七部对小天狼星房间的描写）我想看他会如何开解他的侄孙子，一面告诉他如何在霍格沃兹找到新的人生乐趣，一面又从阿不思那里去了解斯莱特林，理解家庭的意义，或者知晓他哥哥（或者弟弟）雷古勒斯所做的一切。

2.Spare

《哈利·波特与倒霉孩子》中讲述了另一个重点是“spare”，“炮灰”。

书中阿莫斯·迪戈里悲痛地说，伏地魔杀死塞德里克的原话是：Kill the spare.人文的翻译是“干掉碍事儿的”。

阿不思想要穿越时空去拯救塞德里克·迪戈里也是因为这个。塞德里克是一个优秀的青年，输就输在了不是主角。每一个主角背后都有这样的人，他们是略显黯淡的背景，去衬托处决的闪闪发亮。

阿不思说，他觉得自己也是这样的 spare。他活在父亲的阴影之下，永远都是故事的边缘。

这种视角我觉得蛮有趣的。并不是说这些人死去而哈利活着就是他的错，或者一定要通过时空转换把他们救回来（那我的小天狼星和教授怎么办？！），但他们不应该仅仅作为 spare，他们是巨大的牺牲，是值得缅怀的英雄。

塞德里克·迪戈里的设定在很多小说都有，俊朗，杰出，又有点循规蹈矩，不像主角那么复杂且鲜明。这样的角色一般都如同一个 flag，要么会沦为炮灰，要么会黑化。他不会在书中占据大多的篇幅，经历多么扣人心弦的时刻，但他在生活中一定是一个好人，是能够给他的家人和朋友带来安定和温暖的人。

不过我还是觉得，书中把塞德里克·迪戈里设定为受辱后就加入食死徒从而改变历史进程实在太草率了……我觉得塞德里克是不会黑化的，他可能会消沉，但是不会报复。只是个人感觉。

3.

阿不思第一次在 1994 年见到芙蓉·德拉库尔没有什么反应。不过这是他大舅妈，如果有一些补充会好玩一点。

4.

其实这本书最重要的意义，对我来说，在于让我感觉到，原来这个世界依然在运行。

霍格沃兹之战胜利了，对于普通麻瓜来说故事已经结束了。但是对于魔法世界来说，这只是漫长历史的一个节点，生活仍在继续。不管有没有人把它写出来，不管我们从何处知晓那里的故事，都有一群人在魔法世界里生活着，存在着。哪怕不如我们的设想，甚至出现了毁三观的崩坏，哈利·波特和他的朋友们，魔法世界的巫师和女巫们，依然在那个世外桃源自得其乐。

一想到这个，我就会很开心。
