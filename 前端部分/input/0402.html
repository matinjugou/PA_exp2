水母：虽然我很可能伤害你，但是我美呀

图片：Yestone.com 版权图片库

有哪一些高颜值的水母品种？

喵鱼酱，海洋生物科研学徒

谢 @林业杰 oOOo 邀，之前好像有人问过我没来得及回答……

想到几个说几个昂……反正只要放图就行了 23333……

1. 笠花水母 / Flower Hat Jellyfish ;  Olindias formosus

前面 @Heliopora 提到的，顾名思义，就是长得像花帽子一样的水母啦。

隶属水螅纲 Hydrozoa ，淡水水母目 Limnomedusae（该目下也有海生水母）。

2. 银币水母 / Blue Button ;  Porpita porpita

同样属于水螅纲的银币水母事实上是水螅群体，并没有水母体。

3. 僧帽水母 / Portuguese Man O' War（葡萄牙战舰）;  Physalia physalis

隶属水螅纲 Hydrozoa ；管水母目 Siphonophora 。

就是那个最近总出现在朋友圈里的“去海边遇到这个千万不能碰！”，但是其实在国内的话去海边也不一定会碰上，有的公众号还画蛇添足“去青岛的海边遇到这个……”怎么怎么滴，但是它压根没记录在黄渤海出现过，在东海、南海倒有出现过一些。

虽然僧帽水母整个看起来像一只水母，但其实上是一个同时含有水螅体和水母体的群落，每个个体高度分化，不能独立生存。上边那个鼓鼓的是浮囊体。

↓手绘版，图源：Voyage de découvertes aux terres australes : ex...，我加了点注释。

4. 荷包蛋水母（非中文学名） / Mediterranean Jellyfish ; Cotylorhiza tuberculata

从底部看上去，没什么，从上面看——

像不像一个荷包蛋！

隶属钵水母纲 Medusozoa ；根口水母目 Rhizostomae 。

这种水母常见于地中海、爱琴海和亚德里亚海。

5. 咖啡金黄水母 / Northern Sea Nettle ;  Chrysaora melanaster

隶属钵水母纲 Medusozoa ；旗口水母目 Semaeostomeae 。

6. 海月水母 / Moon Jellyfish ; Aurelia aurita

隶属钵水母纲 Medusozoa ；旗口水母目 Semaeostomeae 。

这个在国内的水族馆就比较常见了。是钵水母纲的传统代表。由于个体透明，因此在灯光下结构清晰。中央四个圈是生殖腺，下方是口腕，旁边的小毛毛才是触手。

知乎用户，果壳【珊瑚漫谈】【珊瑚礁漫步】

感谢 @伶俐宝丁 的邀请，头像很可爱所以回答一下。

虽然不是我擅长的方向，但是我的确有心目中的高颜值水母 No.1——花笠水母。

花笠水母的日文名是 ハナガサクラゲ，学名为 Olindias formosus  (Goto, 1903) 。

水螅纲淡水水母目里的小美人。北半球分布在日本本州的中部至九州沿岸。

伞的直径约 10 ～ 15cm ，标准“巴掌脸”~

从名字就可以知道，外形酷似美人所戴花斗笠，粉紫色流苏小坠随流而动，可以说是绰绰身姿水中舞。

可惜的是，本人不在中部至九州，也就在水族馆曾有一面之缘，所以只能借用网络上大师们的美照展示一下其美态。

来源：Flower Hat Jelly (Olindias formosa) | Josh More | Flickr

橙色是生殖腺。像不像斗笠的竹内撑呢~

来源：优秀奖| 科学/自然历史图片故事

来源：圣骑士

此水母虽美，的确带刺（细胞）。裸露的皮肤要是被它的触手碰触到，会有强烈的痛感，但目前没有致人死亡的报告。它用布满刺细胞的触手麻痹捕到小鱼后会整条鱼直接吞下，慢慢消化。另外，这小家伙是底生性的水母，因此很难在水面上看到它，往那水底或者岩石块上找找成功率比较大。为了自身安全，远观勿亵玩之←对其他水生生物也应该这样做。

重要：不管何种水母（除了帕劳那观光名所水母湖里的）或多或少有毒，容易引起过敏反应，严重可致死。自然环境下遇到水母特别是水母群，尽量保持安全距离或者远离。浮潜或水肺潜水尽量减少裸露在外的皮肤面积。

另外想说，我觉得饲养水母过程中看着碟状体慢慢成长成小水母，也是非常治愈的。

私心附上自己以前养水母过程中拍的两张照片~自家海月水母宝宝也很可爱啊。

PS：国内很多旅游景点卖的水母基本上都是小海蜇。无专业设备难以养活，仅供观赏两天就可以倒厕所了。国内海滩上偶尔看到的半透明或带些颜色啫喱状的一滩半固体半液体的玩意大多都是被冲上岸的水母尸体，可以拍照留念问博物，别手痒乱摸，过敏尽快就医不用来知乎提问怎么办。
