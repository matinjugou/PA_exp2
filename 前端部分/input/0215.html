进入管理咨询界和投行前，得先了解哪些硬知识？

图片：skys the limit2 / CC BY

何明科，不写程序的数据工程师不是好产品经理

这个问题在 MBB 等级别的战略管理咨询公司基本是有标准答案的。以 BCG 而言，咨询人员（不了解贵为老板阶层的合伙人，所以不包含在下面讨论内容中）有完整和严谨的评级体系和职业上升通道。因为咨询人员的工作基本是围绕项目的，所以在每个项目之后，该项目的上级和下级会对业务人员的表现进行评分，Hard Skills 和 Soft Skills 是两大基本维度。这里的 Hard Skills，我估计基本对应题主的硬知识。

Hard Skills 主要是解决问题的能力，包括：财务建模（投行常说的「模型」(model) 是什么东西？ - 何明科的回答）、PPT 制作、逻辑分析、数据分析、构造商业假设等等。所以有些答案中将硬知识归结为 PPT 和 Excel，是有一定道理的。Soft Skills 主要是沟通能力和领导力，包括：团队组织、带领和激励，团队沟通，客户管理，客户沟通，项目销售等等。

但是对每个级别的 Hard Skills 和 Soft Skills 要求不一样，大体来说，级别越高，对 Hard Skill 的要求越不明显或者要求越抽象（总不能让 Principal 画 PPT，他们应该更专注在把握大方向和提出最核心的商业假设），对 Soft Skills 的要求越高和越贴近项目销售和回款（管理客户的能力决定客户满意度，影响回款以及续费销售）。

不同级别的顾问有不同的职业发展委员会负责评价体系和职业晋升，比如：Analyst 和 Associate 的职业发展委员会由 Principal 来组织和负责，VP 和 Principal 的由合伙人来组织和负责等等。当顾问的项目数量日积月累到一定程度，职业发展委员会就会将同一个地区同一个级别的咨询顾问在各个项目上得分加总求平均并放到如下的坐标图上，并根据平均值拆分成经典的 2x2 矩阵，最后对不同象限的顾问进行不同程度的职业处理。咨询非常强调培训，如果真的发现某个员工有短板，会定制化地安排相应的培训和提升课程，以及在下一个项目的时候，让项目经理或者合伙人特别关照和辅导式提高。

为了保证每个顾问的诉求有向上反馈的通道（比如：某个项目上表现不好而影响总分，是因为和上级性格不合，或者是因为家里有状况影响返回等等），每个顾问在职业发展委员会上有一个自己的代表（类似于美国各地区的议员），会代表各个顾问在职业发展委员上发言，提出各个顾问的各种诉求。

不带偏见地说，咨询公司强调以人为本也真正做到了，在人员培养、激励、挽留和发展的制度上做得真心非常不错，从系统、制度到文化。根据 Glassdoor（美国权威的招聘网站，以收集内部员工的真实反馈及薪酬待遇而闻名于世）发布的 2015 年英美最佳雇主榜单，前十黄金广告位有三席被 MBB 占据：Bain、BCG 和 McKinsey 分别占据第 2、第 5 和第 9 名。而且过往年份的排行也大致如此，BCG 常年排在第二第三的位置且是管理咨询公司中的最高位置。
