有时也会迷惑，我到底是来种地，还是来读硕士的

图片：Yestone.com 版权图片库

作为农业类硕士，下地干活难以避免，怎样平衡科研中脑力劳动与体力劳动的比例，来体现一名研究生该有的价值？

Sost Green，我是一个种地的。

首先，很高兴你能够认识到，作为农业类的研究院，下地干活难以避免。作为一位优秀的电工，你总会有几个月剥线头的经历。作为一位优秀的农业工作者，你总会有几个月田间工作的经历。第一个问题是，这么做的意义是什么？显然不是和一线工人比工作日常工作能力。在他们的工作岗位，你完全没有性价比。

我愿意和你分享几个故事：

1、几年前有个研究生送到我们基地做实验，小姑娘，很漂亮。课题大约是有机肥料和无机肥料对单头菊花生产的效果比较。单因子实验，分了 3 个水平：纯鸡粪，鸡粪 + 硝酸钙，鸡粪 + 磷酸二氢钾，问我们买了一些鸡粪，搞了一些硝酸钙和磷酸二氢钾。重复数量不清楚。指标不记得了，好像因为赶时间，测量指标不会等到采收期结束。应该有些人能够看出来，逻辑相当混乱。当时情况确实是那样的。

我当时其实不负责单头菊生产，但是看她的实验有些捉急。她所做的实验，和我们实际生产完全联系不上。于是有一次碰上了，就和她聊了一些问题：

  • 有机肥的背景她是否清楚？
  • 单头菊实际生产过程是否明确？
  • 水平选择是否合理？
  • 几个水平完全可以单独作为因子，分别建立水平，这样是不是对生产意义更大？
  • 我们实际生产中是采用肥水灌溉，她的实验仍然采用大田自然生长方式，管理方式是否合适？
  • 她几周才会来一次，病虫害防控很难把握。

整个沟通的过程中对她人畜无害的呵呵记忆犹新，但是很显然这些问题她最终也没有关注。最后不知道她的论文怎样了？

2、我刚毕业时，在做温室计算机控制，园艺管理员是一个中农的女硕士。我们同时入职的。我的工作就是按照她的要求在计算机设定肥水灌溉浓度，并完成灌溉。有一天她过来看我工作，她当天的灌溉 EC 要求是 1.2 ms/cm。计算机显示实时值在 1.1-1.2 范围波动。她看了一会儿，很激动的和我说。她要的是 1.2 ms/cm。为什么有时是 1.1？她就要 1.2，不接受任何系统波动。否则如果花种不好，就是我灌溉不对……

通过这两个例子，我想告诉你的是：作为一位合格的农业工作人员，无论你是生产技术员，还是研究院，还是周边服务员，你首先必须了解生产过程。农业的问题源自生产，必须以生产的角度来解决。否则你的任何想法，行为，决定只是空中楼阁，完全不接地气。就会做出完全没有意义的科研课题，提出完全没有道理的生产要求。所以说，你必须有足够的田间工作的经历，才能了解生产过程。

作为农业研究人员，了解生产过程只是研究过程的第一步。当你在生产过程中发现问题，分析问题、设计试验。你的研究课题就能够深入展开了。这个过程，就需要运用脑力劳动，完全发挥出你的真实价值。当你作出了优秀的试验设计后，又需要在现有生产工艺的背景下去验证试验。这又是一个体力劳动的过程。

回顾上述 3 个步骤，对于研究生阶段的研究员来说。我认为首先还是要有一定的生产过程积累。特别是对正常成功生产模式的了解。很多研究生，一开始就没有建立起正确的认识。打个比方，我们在温室内采用无土栽培，番茄每平米产量可以达到 85kg。如果你还在温室内采用常规土壤栽培，连 10kg 都困难，这样的研究就完全没有任何意义了。因为，就算你有任何结论，这温室我连本钱都回不来，直接就赔死掉了。

这个成功的生产过程不是简单的每天灌多少水，施多少肥，打多少药。而是一个对周年不同情况的特殊调整。这会是一个种植企业几十年的生产经验积累，研究人员的一个价值就在于可以更快速的或许并且更容易融合，实现创新。

综上，针对你的问题，我认为作为一位合格农业的研究员，最好的平衡脑力和体力劳动最好的办法是遵循合理的个人发展步骤，首先了解生产，再挖掘生产过程中的问题，动用大脑不断去解决。这其实也是你一辈子去做的事情。
